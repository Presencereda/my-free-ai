# Laith-Sentinel 🦁🛡️

### The BitVM Logic Integrity Framework

**Laith-Sentinel** is a specialized security project focused on protecting Bitcoin Layer 2 protocols from "Logic Collapse" vulnerabilities. 

### Why this project?
I discovered a **Critical CVSS 10.0** vulnerability in the BitVM-based project "Clementine" (GHSA-fg26-7pfh-r7wq). This discovery proved that off-chain state transitions can be manipulated if not strictly bound to ZK proofs.

### Goals:
* Build automated scanning tools for Rust-based BitVM logic.
* Develop security standards for state-diff postprocessing.
* Protect the future of Bitcoin L2s for the next generation.
